# 缺陷报告

## 一、缺陷概述

本次测试共发现**21个缺陷**，按严重程度分类：
- **严重缺陷**：5个
- **一般缺陷**：12个
- **提示缺陷**：4个

---

## 二、详细缺陷列表

### 缺陷ID：BUG-001

| 项 | 内容 |
|----|------|
| 缺陷标题 | 登录接口密码明文传输和存储 |
| 严重程度 | 严重 |
| 优先级 | 高 |
| 所属模块 | 登录接口 |
| 前置条件 | 系统正常运行 |
| 缺陷描述 | 用户密码以明文形式在网络传输和数据库存储，存在严重安全隐患 |
| 预期结果 | 密码应进行加密传输，数据库存储MD5或BCrypt加密后的密文 |
| 实际结果 | 密码明文传输，数据库中user_password字段存储明文 |
| 重现步骤 | 1. 抓包登录请求，可看到明文密码<br>2. 查询数据库user表，看到明文密码 |
| 测试环境 | JDK17 + MySQL |
| 备注 | 安全合规性问题，可能导致用户信息泄露 |
